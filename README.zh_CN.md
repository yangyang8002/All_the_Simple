## 一、整合包定位

做这个包的想法挺简单：打开就能玩，不用调一堆配置；路线自由，不硬拽着你肝。  
概括起来就三点：

- **开箱即玩**：下载、解压、启动，剩下的事包都帮你办好了。
- **不卡你不拖你**：Create 机械动力是主轴，工业、魔法、农业三条线想怎么走就怎么走，没有阴间配方故意拖时间。
- **单人能毕业，联机更好玩**：自己慢慢折腾也能到头，拉上朋友更热闹。内置语音、QQ 群 bot、FTB 团队系统都准备好了。

## 二、核心特色

### 1. 机械动力全家桶

Create 带上三十几个扩展，挖矿、附魔、能源到龙息发电，一条完整的机械科技线直接铺好。  
组装流水线的乐趣，这里管够。

**主要扩展模组：**
- Create: Dragons Plus —— 龙类自动化
- Create: Enchantment Industry —— 附魔自动化
- Create: New Age —— 能源与发电
- Create: Ore Excavation —— 矿物开采
- Create: Copycats+ —— 建筑复制
- 更多内容留给你自己探索……

### 2. 轻量魔法

新生魔艺、神化和魔法使搭在一起，法术直接合成，不卡灵力、不卡阶段。附魔等级我们也重新调过，拿到材料就能一直往上升，玩的就是一个痛快。

### 3. 快乐农业

农夫乐事、神秘农业加上无中生有：神赐，凑出一条从种菜、做饭到吃 buff 的全自动厨房。机器一摆，食物成箱出，背包再也不用塞满吃的。

### 4. 观光与冒险

- **YUNG 系列结构优化**：原版的矿洞、要塞、神殿全给翻修了一遍，养眼又值得探。
- **四条 Boss 线等你打**：
  - 冰火传说社区版（Ice & Fire: Community Edition）
  - 灾变（L_Ender's Cataclysm）
  - 暮色森林（Twilight Forest）
  - 永恒星光（Eternal Starlight）
- Boss 掉的破烂别扔，Create 产线和 ProjectE 都能回收再利用。
- 地形丰富到爆，超多生物群系（Biomes O' Plenty、Oh The Biomes We've Gone）加上 Macaw 全套建筑方块，喜欢逛图造景的人有福了。

### 5. 性能 & 体验

Sodium 全系列优化、FerriteCore 内存治理、MemorySweep 清垃圾，还给粒子加了异步处理。亲测分 4 GB 内存，光影能流畅跑起来。

### 6. 完全本地化

自动汉化、拼音搜索、输入法冲突全部修好，搜物品敲中文就行，不用来回切输入法。

### 7. 多模组联动

- L2 系列（L2Complements、L2Hostility、L2Weaponry）—— 额外挑战
- Industrial Foregoing —— 更多工业自动化
- Applied Energistics 2 —— 现代存储与物流
- Mekanism —— 高产科技线
- ProjectE —— 等价交换，给你养老的底气

## 三、玩法路线

### 🔧 机械工程师

```
铜 → 锌 → 黄铜 → 火车 → 飞艇 → 龙力发电机
```
**最终目标**：用 Create 的龙+模块全自动击杀末影龙。

### 🧙 魔法学者

```
新生魔艺 → 法术自动化 → 神化无限附魔 → 魔法使灵装
```
**最终目标**：一发威力拉满的“湮灭光束”。

### 🌾 农场大亨

```
神秘农业 6 级精华 → 农民乐事全套厨房 → ProjectE 等价交换
```
**最终目标**：资源完全自由，提前养老。

### 📐 生电玩家

投影 + Create + Custom Machinery

**最终目标**：别问，生电人的毕业日永远是明天。

## 八、版本说明

| 端 | 命名规则 |
|----|----------|
| 客户端 | Release/Beta/Alpha + 版本号 + c/hd (+ fix + 修复号) |

> **hd**：硬核版，包含莱特兰-恶意（L2）及其附属
